yml配置文件注意事项

YAML 文件格式对缩进和语法要求比较严格，类似于 Python 的缩进规则。反正最好在**”-“和“:”**之后添加一个空格再输入值

YAML 使用空格进行缩进，不允许使用制表符（tab），通常推荐使用两个空格进行缩进

  • 键和值之间需要用冒号和一个空格分隔 –> key: value
  • 列表项需要用破折号（-）和一个空格前缀
